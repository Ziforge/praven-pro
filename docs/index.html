How It Works

1

Upload BirdNET CSV

Upload your BirdNET detection results (works with BirdNET Analyzer, BirdNET-Pi, and other variants).

2

Enter GPS Coordinates

Provide latitude and longitude of your recording location. Habitat and weather will be auto-detected.

3

Automatic Validation

Praven applies biological rules: temporal patterns, habitat matching, geographic range checks, and taxonomic constraints.

4

Review Results

Get validated detections: auto-accepted (high confidence), auto-rejected (violations), and priority review (top 3 per species).

Installation

üì¶ PyPI Installation

Install via pip (recommended):

pip install praven-pro

Run validation:

praven detections.csv \
  --lat 63.341 --lon 10.215 \
  --date 2025-10-15

üêç From Source

Clone and install:

cd praven-pro
pip install -r requirements.txt

Run validation:

  --lat X --lon Y \
  --date YYYY-MM-DD
